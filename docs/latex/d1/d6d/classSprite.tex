\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSprite}{Sprite}} class\+: \mbox{\hyperlink{classSprite}{Sprite}} is a class that represents the rendering properties of a Component. 

The \mbox{\hyperlink{classSprite}{Sprite}} class manages the graphical representation of a Component using SFML.
