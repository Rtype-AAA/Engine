Use the following command to compile the engine\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/Bbuild}
\DoxyCodeLine{make -\/Cbuild}

\end{DoxyCode}


Use the following command to compile the engine and its tests\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/Bbuild -\/DBUILD\_TESTS=ON}
\DoxyCodeLine{make -\/Cbuild}

\end{DoxyCode}


Use the following command for create the package (.tgz or .zip) after compile\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cd build}
\DoxyCodeLine{cpack}

\end{DoxyCode}
